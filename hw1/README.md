# How to run the program
1. Execute the instructions below :
    ```
    conda activate habitat
    python3 bev2front.py
    ```
2. You'll see a small window displaying a BEV image.
3. Use your mouse to click 4 points in either clockwise or counterclockwise order.
4. Press any key when you finish defining the region, then you will see the projected polygon in the front view image.
